# Shortcuts

| Keys       | Action                       |
|------------|------------------------------|
| `l` / `L`  | change palette               |
| `s` / `S`  | change shader                |
| `v` / `V`  | change accelerator           |
| `x` / `X`  | change position setter       |
| `r` / `R`  | change matrix generator      |
| `p`        | set positions                |
| `t`        | set types                    |
| `m`        | set matrix                   |
| `w`        | toggle space wrapping        |
| `SPACE`    | pause physics                |
| `F11`      | toggle full screen           |
| `ALT`+`F4` | quit                         |
| `+` / `-`  | zoom                         |
| `z`        | reset zoom                   |
| `Z`        | reset zoom (fit window)      |
| `a`        | toggle advanced GUI          |
| `c`        | toggle traces (clear screen) |
| `h`        | hide GUI / show GUI          |
